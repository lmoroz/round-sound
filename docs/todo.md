в этом репозитории нам надо создать мини-десктопное приложение для windows на основе wails+vue.js+vite+ts
только версии vite и vue.js, ts + lucide icons , я уже сделала шаблон для установки frontend в @frontend 

суть приложения — виджет, отображающий, что сейчас играет в медиаплеерах или в браузере на разных источниках (браузеры используют плагин @WebNowPlaying ) 

виджет должен выглядеть так: :

@round-sound-preview.jpg 

прозрачное окно, вся область окна — перетаскиваемая, запоминает своё положение между перезапусками приложения,
круглый интерфейс, 
внутри на подложке — обложка, которую передаёт медиа-сервис или браузерный плагин,
над обложкой — информация о треке (название трека, артист, альбом) и кнопки like, prev, shuffle, play/pause, next repeat, dislike — их нажатие и статус должны передаваться в медиа-сервис или в плагин браузера
по кругу виджета идёт прогресс воспроизведения текущего трека
снаружи круга — «лучи» audio levels с loopback
